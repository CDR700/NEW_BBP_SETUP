# NEW_BBP_SETUP
A simple tool i made to assist me in the creation of Bug Bounty programs folders.

you need a folder in your home dir, named "BugBounty"

### Install
```
mkdir ~/BugBounty
chmod +x newbbp 
sudo mv newbbp /usr/local/bin/

```
